ipsvd - installation

ipsvd installs into /package. If you don't have a /package directory, create it now: 
 # mkdir -p /package
 # chmod 1755 /package
Download ipsvd-0.12.0.tar.gz into /package and unpack the archive 
 # cd /package
 # gunzip ipsvd-0.12.0.tar
 # tar -xpf ipsvd-0.12.0.tar
 # rm ipsvd-0.12.0.tar
 # cd net/ipsvd-0.12.0
If you are installing on Linux or MacOSX, and want to make the sslsvd and sslio programs available, download the matrixssl library version 1.7.1 (newer versions probably work too) and copy the archive to the src/matrixssl.tar.gz (alternatively download from the Debian archive) 
 # cp matrixssl-1-7-1-open.tar.gz src/matrixssl.tar.gz
Now compile and install the ipsvd programs 
 # package/install
If you want to make the man pages available in the /usr/local/man/ hierarchy, do 
 # package/install-man
To report success: 
 # mail pape-ipsvd-0.12.0@smarden.org <compile/sysdeps
If you use ipsvd regularly, please contribute to the project.

Refer to the examples to learn how to set up services with ipsvd.
