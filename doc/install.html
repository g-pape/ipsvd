ipsvd - installation

ipsvd installs into /package. If you don't have a /package directory, create it now: 
 # mkdir -p /package
 # chmod 1755 /package
Download ipsvd-0.8.0.tar.gz into /package and unpack the archive 
 # cd /package
 # gunzip ipsvd-0.8.0.tar
 # tar -xpf ipsvd-0.8.0.tar
 # rm ipsvd-0.8.0.tar
 # cd net/ipsvd-0.8.0
Compile and install the ipsvd programs 
 # package/install
If you want to make the man pages available in the /usr/local/man/ hierarchy, do 
 # package/install-man
To report success: 
 # mail pape-ipsvd-0.8.0@smarden.org <compile/sysdeps
