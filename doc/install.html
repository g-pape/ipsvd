ipsvd - installation

ipsvd installs into /package. If you don't have a /package directory, create it now: 
 # mkdir -p /package
 # chmod 1755 /package
Download ipsvd-0.9.2.tar.gz into /package and unpack the archive 
 # cd /package
 # gunzip ipsvd-0.9.2.tar
 # tar -xpf ipsvd-0.9.2.tar
 # rm ipsvd-0.9.2.tar
 # cd net/ipsvd-0.9.2
If you are installing on Linux, and want to make the sslio program available, download the matrixssl library version 1-0-2 and move it into the src/ directory (alternatively download from the Debian archive) 
 # mv matrixssl-1-0-2.tar.gz src/
Now compile and install the ipsvd programs 
 # package/install
If you want to make the man pages available in the /usr/local/man/ hierarchy, do 
 # package/install-man
To report success: 
 # mail pape-ipsvd-0.9.2@smarden.org <compile/sysdeps
If you use ipsvd regularly, please contribute to the project.

Refer to the examples to learn how to set up services with ipsvd.
