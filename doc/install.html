ipsvd - installation

Download ipsvd-0.1.0.tar.gz and unpack the archive 
  # gunzip ipsvd-0.1.0.tar
  # tar -xpf ipsvd-0.1.0.tar
  # cd ipsvd-0.1.0
Download the dibdns client library, unpack the archive, and build the library 
  # gunzip djbdns-pd-1.05.tar
  # tar -xpf djbdns-pd-1.05.tar
  # cd djbdns-pd-1.05
  # make
  ...
  # cd ..
Compile and install the ipsvd programs 
  # cd src
  # make
  ...
  # install -m0755 tcpsvd /usr/local/bin/tcpsvd
  # install -m0755 udpsvd /usr/local/bin/udpsvd
  # install -m0755 ipsvd-cdb /usr/local/bin/ipsvd-cdb
  # cd ..
If you want to make the man pages available in the /usr/local/man/ hierarchy, do 
  # gzip man/*.8
  # install -m0644 man/*.8.gz /usr/local/man/man8/
