Name

udpsvd - UDP/IP service daemon

Synopsis

udpsvd [-vh] [-u user] host port prog

Description

Options

host
host either is a hostname, or a dotted-decimal IP address, or 0. If host is 0, udpsvd accepts connections to any local IP address.
port
udpsvd accepts connections to host:port.
prog
prog consists of one or more arguments.
-u user
drop permissions. Switch user ID to user's UID, and group ID to user's primary GID after creating and binding to the socket.
-v
verbose. Print verbose messaged to standard out.
-h
lookup hostname.

See Also

tcpsvd(8), ipsvd-cdb(8)

http://smarden.org/ipsvd/
