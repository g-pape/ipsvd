Name

tcpsvd - TCP/IP service daemon

Synopsis

tcpsvd [-dEHv] [-c num] [-u user] [-r dir | -x cdb] host port prog

Description

Options

host
host either is a hostname, or a dotted-decimal IP address, or 0. If host is 0, tcpsvd accepts connections to any local IP address.
port
tcpsvd accepts connections to host:port.
prog
prog consists of one or more arguments. For each connection, tcpsvd runs prog, with descriptor 0 reading from the network and descriptor 1 writing to the network. By default it also sets up TCP-related environment variables, see tcp-environ(5)
-c num
concurrency. Handle up to num concurrent connections. Default is 30.
-u user
drop permissions. Switch user ID to user's UID, and group ID to user's primary GID, before starting prog to handle the connection.
-r dir
rules directory.
-x cdb
rules cdb.
-d
deny.
-E
no ucspi environment. Do not set up TCP-related environment variables.
-H
no remote hostname lookup.
-v
verbose. Print verbose messsages to standard out.
